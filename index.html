CAMCO

Castle Age Monster Code Opener

View project onGitHub

Castle Age Monster Code Opener, "CAMCO" is a handy tool to open monster code in CA. It converts monster code of CA to monster URL of web3 or facebook page, and opens it for you.

Usage: 1. Select the monster code. 2. Right click the selection. 3. Check the context menu "Castle Age Monster Code Opener" > open or copy the monster URL in either way.
